---- ## Authentication ### Overview Konker Platform API uses the OAuth protocol to allow users securely access data. Before making REST API calls, you must get a access token using [OAuth 2.0](http://oauth.net/). You can see the API endpoints documentation and interact with them on this page. ### Get OAuth 2 Access Token * Grant Type: Implicit Grant (or User Agent) Flow * Access Token URL: http:///v1/oauth/token * Client ID: your e-mail * Client Secret: your password ### Swagger Authorization To use Swagger endpoints: * Login * Use your email and password * Click the 'Authorize' button in upper right conner ---- ## Concepts ![Concepts](static/diagram.png) ----
